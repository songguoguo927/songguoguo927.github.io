函数详细讲解

基础知识

function
1.“洗衣机”，具备某个功能的一个方法。方法本身无意义，执行了才有价值。
2.两部分组成
创建: function 方法名(){//函数体:存放的是某个功能实现的js代码}
1.首先开辟一个新的内存空间，浏览器为其分配一个16进制的内存空间eg，xxxfff000，
2.把函数体里的js代码当成字符串存进内存空间
3.再把内存空间的地址名赋给当前函数名
执行:方法名();创建一个函数可多次执行，且互相执行之间没有关系。
1.sum；表示当前函数本身；sum();表示执行当前函数，函数执行的目的:把函数之前存储的那些字符串变为js代码，然后让js代码从上之下依次执行。
2.函数在执行时，形成一个私有作用域供函数体中代码执行
3.把函数体中的代码字符串变为代码从上到下执行
3.实例:两个数（固定）求和的函数
null 挖坑啥没干（推荐）， 0坑都没挖。
4.闭包:函数执行时形成的私有作用域把我们函数体中的代码都包起来了，保护了里面的私有变量不受外界干扰（外面获取不到，修改不了），我们把函数执行时形成的这种保护机制叫做闭包。
形参（函数暴露的入口）
实例，求任意两个数之和。sum(num1，num2);
return（当前函数提供的出口）
1,函数体中没有return，由于闭包机制在方法外拿不到里面的total，默认返回值undefined。
2, 有return，return total;返回给外面的是total变量存储的值，不是变量total。要想知道具体的值多少，还得在外边var total=sum(100,200);定义一个变量来接收返回值，此次total不是里面的total，之所以可以重名是因为里面的由于闭包机制传不到外面。所以外面可以这样命名。
arguments
1.函数天生自带的（不管写没写形参，也不管是否传值，arguements始终存在）用来接收函数传递进来的参数值的一个集合（传几个存几个）。
2.是一个object数据类型的值，以数字作为索引从0开始，类数组，有length。
3.实例:“任意数求和”:给我任意个数，求出它们的和。
关键点:具体几个数不知道，没办法定义形参的个数。利用arguments.length循环类数组集合。
优化1:把传进来的值转换为number类型，防止出现字符串拼接
优化2:对非有效数字不累加，防止结果为NaN（isNaN（）判断）
匿名函数:没有起名字的函数(-相反有函数名字的叫实名函数)
js中常用两种方式
1.自执行函数（也叫立即执行函数）:定义执行一起
;（function（num）{}）（100）;//第一个分号作用为了防止它上面的代码结束不加。分号还可以用+，-，！，~替换（装×用）。
2.函数表达式:把函数定义的部分当成一个值赋值给一个变量或者元素的某一个行为
oDiv.onclick=function(){}
