函数详细讲解

基础知识

function

1.“洗衣机”，具备某个功能的一个方法。方法本身无意义，执行了才有价值。
2.两部分组成
创建: function 方法名(){//函数体:存放的是某个功能实现的 js 代码}
1.首先开辟一个新的内存空间，浏览器为其分配一个 16 进制的内存空间 类似xxxfff000。
2.把函数体里的 js 代码当成字符串存进内存空间
3.再把内存空间的地址名赋给当前函数名
执行:方法名();
创建一个函数可多次执行，且互相执行之间没有关系。
1.sum；表示当前函数本身；
sum();表示执行当前函数，函数执行的目的:把函数之前存储的那些字符串变为 js 代码，然后让 js 代码从上之下依次执行。
2.函数在执行时，形成一个私有作用域供函数体中代码执行
3.把函数体中的代码字符串变为代码从上到下执行

4.闭包:函数执行时形成的私有作用域把我们函数体中的代码都包起来了，保护了里面的私有变量不受外界干扰（外面获取不到，修改不了），我们把函数执行时形成的这种保护机制叫做闭包。

形参（函数暴露的入口）
实例，求任意两个数之和。sum(num1，num2);

return（当前函数提供的出口）
1,函数体中没有 return，由于闭包机制在方法外拿不到里面的 total，默认返回值 undefined。
2, 有 return，return total;返回给外面的是 total 变量存储的值，不是变量 total。要想知道具体的值多少，还得在外边 var total=sum(100,200);定义一个变量来接收返回值，此次 total 不是里面的 total，之所以可以重名是因为里面的由于闭包机制传不到外面。所以外面可以这样命名。

arguments
1.函数天生自带的（不管写没写形参，也不管是否传值，arguements 始终存在,用来接收函数传递进来的参数值的一个集合（传几个存几个）。
2.是一个类数组对象，以数字作为索引从 0 开始，类数组，有 length。

实例:“任意数求和”:给我任意个数，求出它们的和。

关键点:具体几个数不知道，没办法定义形参的个数。利用 arguments.length 循环类数组集合。
优化 1:把传进来的值转换为 number 类型，防止出现字符串拼接
优化 2:对非有效数字不累加，防止结果为 NaN（isNaN（）判断）

匿名函数:没有起名字的函数(-相反有函数名字的叫实名函数)
js 中常用两种方式
1.自执行函数（也叫立即执行函数）:定义执行一起
;（function（num）{}）（100）;//第一个分号作用为了防止它上面的代码结束不加。分号还可以用+，-，！~替换（装 × 用）。
2.函数表达式:把函数定义的部分当成一个值赋值给一个变量或者元素的某一个行为
oDiv.onclick=function(){}
